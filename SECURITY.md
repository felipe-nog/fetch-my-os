# Security Policy

## Supported Versions

Currently, the following versions of `fetch-my-os` are being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability in `fetch-my-os`, please report it directly to the author by emailing **felipenogueira.tec@gmail.com**.

**What to expect:**
* You should receive an initial response to your report within 48 hours.
* If the vulnerability is confirmed, a patch will be developed and a new version will be released as soon as possible.
* You will be kept informed of the progress and, with your permission, credited in the release notes or advisory once the fix is public.
